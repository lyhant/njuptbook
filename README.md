# NJUPT IS NOT ENOUGH

> 只有每天争取自由和生存者，才配享受自由和生存。于是少年、壮年和老年人不惧风险，在这里度过有为的年辰。

如果这个文档可以给学弟学妹们带来一些方便，那么这本书的目的就已经达到了！我们没有一片汪洋的思想，但是我们有在南邮许多年的故事，而这些故事会和你重叠，当你和我们一样面临抉择的时候，请做出超过我们的选择。
